# SwarmConductor
SwarmConductor Строгий фреймворк оркестрации ИИ-агентов.  Запрещает LLM кодить напрямую, внедряя жесткий пайплайн: Разведка > План > Код > QA.  Внутри: Gatekeeper: Строгий пул из 7 субагентов.  Среды: Конфиги для Antigravity IDE и CLI (Claude Code, Cursor).  Конец ИИ-самодеятельности.
